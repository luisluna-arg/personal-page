# Luis Luna - Personal Page

A personal landing page showcasing my professional profile, skills, and ways to connect.

## About

This is my personal website built as a single-page React application. It features:

- **Header** - Introduction and welcome section
- **About Me** - Professional background and skills overview
- **Connect With Me** - Links to GitHub and LinkedIn profiles
- **Footer** - Contact information

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **Bootstrap 5** + **React Bootstrap** - Styling and components
- **Docker** - Containerized deployment with Traefik reverse proxy
